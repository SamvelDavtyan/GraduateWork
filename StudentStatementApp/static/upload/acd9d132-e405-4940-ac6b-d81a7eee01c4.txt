Разработка информационной системы для осуществления контроля качества предоставляемых услуг и обработки нештатных ситуаций в ООО "Система ПБО"

Цель: разработать приложение WPF для осуществления аудита, проверок отдельных точек сети быстрого питания с возможностью формирования отчетов таких проверок со стороны головного предприятия и составления отчетности по закупкам, оборудованию, их состоянию отдельными точками также с возможностью формировать заявки на замену оборудования, изменение локального поставщика, и отчетность по иным нештатным ситуациям.

Пользователи: сотрудник головного предприятия (которое мониторит точки), ответственный от точки (руководитель)

Возможности сотрудника головного предприятия:
- Внести данные о новой открывшейся точке (название, описание, юр. адрес, физ. адрес, руководитель, данные помещения)
- Добавить, изменить, удалить данные об оборудовании и иных ресурсов точки
- Внести данные о поставщиках продуктов для точки*
- Создать план проверок для каждой точки (должен быть виден руководителю точки)
- Создать перечень замечаний по проверке (с указанием описания, типа замечания)
- Создать отчет с перечнем замечаний и комментарием, указать срок устранения замечаний
- Рассмотреть тикет руководителя точки и вынести решение (выдать новое оборудование, изменить поставщика) и изменить статус такого тикета
- Возможность посмотреть статистику по количеству замечаний к точке с целью улучшить качество предоставляемых услуг. При этом можно создать "рейтинг" точек и работать с самыми "низкорейтинговыми"

Возможности руководителя точки:
- Внести данные о поставке и продуктах поставленных на точку в определенный период с указанием времени, и перечнем продуктов с указанием срока годности
- Просмотреть план проверок для точки
- Просмотреть перечень оборудования и иных ресурсов для точки
- Просмотреть отчеты проверок и отдельные замечания и назначить ответственных за их устранение
- Изменить статус замечания (напр. со статуса "Назначено" на "Ожидает доп. проверки")
- Создать тикет о замене/починке оборудования либо замечании о качестве поставляемых продуктов
